# Z dychom swjatoh Jana

```
Z dychom swjatoh Jana


kćěje lubozna,


połna róžow rjana,


róžow kralowna.



R.: Aj, ty róža najrjeńša, 


kaž ta moja holička.


Aj, ty róža najrjeńša, 


kaž te naše holiča!
```


```

Z dychom swjatoh Jana

Z dychom swjatoh Jana
kćěje lubozna,
połna róžow rjana,
róžow kralowna.

R.: Aj, ty róža najrjeńša, 
kaž ta moja holička.
Aj, ty róža najrjeńša, 
kaž te naše holiča!

Makojca so šěri 
połna z hordosću,
słabje pak so měri
k twojom runjeću.

Nalika so puka
z hłójčku šěrokej,
podarmo pak kuka,
hač ći runja njej'

```
