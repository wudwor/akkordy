# Nad morjom stejitaj hrodaj

```

Nad morjom stejitej hrodaj,


dwaj hrodaj wysokej.


We jednym rjane holico,


we druhim rjany holc.


Wonaj so lubo mejestaj,


hromadu princ pak njemozestaj.


Won da swojej lubce napisac,


zo chce ju prez morjo wzac.

```


```
Nad morjom stejitaj hrodaj

Nad morjom stejitej hrodaj,
dwaj hrodaj wysokej.
We jednym rjane holico,
we druhim rjany holc.
Wonaj so lubo mejestaj,
hromadu princ pak njemozestaj.
Won da swojej lubce napisac,
zo chce ju prez morjo wzac.

Ha lubk je na morjo wujel,
na morjo hluboke.
Za sydom dnjow je tam dojel,
hdzez lubka cakase.
Wonaj sej rucce zawdastaj,
hromadze na colm poskocistaj.
Won z pazomaj wjesle zahiba
a borze sej dojedze kraj.


```
