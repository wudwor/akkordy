```

Naša Hanka w wěncu steji 


zelenym kaž brěza w meji.

abo je snadź kmótřička?

```


```

Naša Hanka w wěncu steji 

Naša Hanka w wěncu steji 
zelenym kaž brěza w meji.
Je dźě wona njewjesta 
abo je snadź kmótřička?

Njewjesta je Hanka naša.
Wěš ty, štó so za njej praša?
Jedyn młodźenc šikwany,
pěkny, bohabojazny.

Dźitaj, dźitaj, žohnowanaj,
wjele zboža přeju wamaj,
zo byštaj wój za lĕtko
wokoš’wałoj hólčatko!


```
