# Pod Kulowom we holi

```
F
Pod Kulowom we holi
                    C
najrjeńši wrjós tam mam.
F           B
Mje wutroba boli pak,
C                 F
zo mam šćipać jón sam.

    F
R.: Hdy by tam ze mnu šła,
   B            C
za wrjosom hladała,

to rjeńši by snadź kćěł,

ja bych će lubo měł.
```
