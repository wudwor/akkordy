# Anička, dušička, kde si bola

```

Anička, dušička, kde si bola, 


keď si si čižmičky zarosila.


[:Bola som v hájičku, žala som trávičku,


duša moja, duša moja.:]

```


```
Anička, dušička, kde si bola

Anička, dušička, kde si bola, 
keď si si čižmičky zarosila.
[:Bola som v hájičku, žala som trávičku, 
duša moja, duša moja.:]

A ja som po tri dni trávu kosil, 
ja som si čižmičky nezarosil.
[:A ja som hrabala, teba som čakala,
duša moja, duša moja.:]



```
